# Overview

The Medical Insurance Dashboard presents key metrics and visual analytics for insurance data, allowing users to quickly identify trends and cost drivers.

## Project Objects:

-Analyze drivers of medical insurance costs

-Visualize health and demographic patterns

-Build an interactive analytics dashboard

-Demonstrate data storytelling with healthcare data

## Key Performance Indicators (KPIs):

*Average patient age

*Total medical charges

*Total number of patients

*Charges distribution across different demographic and health factors

-These insights support data-driven decisions for insurance risk analysis and healthcare cost management.

## Interactive Filters:

*BMI Category

*Gender

*Smoking Status

*Region 

-This enables customized analysis of insurance charges across different patient groups

## Used Tools

*Microsoft Excel

*Data cleaning

*Power Query & Pivot
